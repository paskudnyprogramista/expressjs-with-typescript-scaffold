## Getting started

Clone git repository

```bash
$ git clone --depth=1 git@github.com:paskudnyprogramista/expressjs-with-typescript-scaffold.git <app-name>
```

Install dependencies

```bash
$ cd <app-name>
$ npm install
```

Update `package.json` file with your data.

## Run production

```bash
$ npm run start
```

## Run development

```bash
$ npm run dev
```
